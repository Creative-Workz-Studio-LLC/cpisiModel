== 1. The Mathematical Anchor (The Lucas Sequence)
All dimensions must be factors of **11**. We reject the "Standard 8px Grid" of the Babylonian web.

*   **1U (11px):** The base unit of breath (Padding/Margins).
*   **2U (22px):** Secondary separation.
*   **3U (33px):** Header height, Sidebar width, Terminal actuators.
*   **4U (44px):** Overlay padding, Major section gaps.
*   **8U (88px):** Large jurisdictional boundaries.

== 3. Typography & Density (The 2.5K Standard)
Optimization target: **2560x1600**.

*   **Root Font:** 8px - 11px. No larger.
*   **Density:** Content must wrap tightly. We reject horizontal "Web Bloat."
*   **Writing Mode:** Sidebars use `vertical-rl` to preserve horizontal tension.

== 4. Block-Based Unity (Hand-in-Hand)
*   **Jurisdiction Naming:** UI labels must match directory names (e.g. GATE folder = GATE navigation).
*   **Surgical Edits:** Only edit the specific block required. No wholesale overwrites once the foundation is set.
